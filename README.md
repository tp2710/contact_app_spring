# contact_app_spring
Contact Application using Spring

#Technology Stack:

1. Java SE
2. Java EE - JSP, Servlet
3. Spring Web MVC
4. MySQL5.x
5. Apache Tomcat 7.x
6. HTML , CSS, JavaScript
7. AJAX, jQuery
8. Apache Maven

# Contact Application Requirement

+ Multi user application
+ Roles : User & Admin
+ User Registration
+ Login - User & Admin
+ Logout - User & Admin
+ Contact - Add
+ Edit Contact
+ List Contact
+ Search Contact (free text)
+ Delete Contact - One By One
+ Delete Contact - Bulk
+ User and Admin - Profile Edit
+ Contact Export/Import - PDF, XLS, VCF, CSV
+ Admin can see all registered users
+ Admin can Block or Active any user
+ Basic Security
